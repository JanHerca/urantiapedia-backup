Este editor Visual es un editor WYSIWYG (Lo que ves es lo que se obtiene) basado en el popular [CKEditor 5](https://ckeditor.com/).

En la captura de pantalla se muestran los controles más destacados del Editor Visual:
1. Título de la página
2. Botonera de guardado:
    - Botón **Guardado / Guardar**. Si pone **Guardado** es que no hay cambios; si pone **Guardar** haz click para hacer los cambios permanentes. **¡Las páginas no se auto-guardan por lo que no olvides de guardar frecuentemente!** :point_left:
    - Botón **Página**: permite volver a modificar los metadatos de la página, como el título, descripción, y etiquetas.
    - Botón **Cerrar**: cuando has terminado con la página cierra aquí para volver al modo de lectura.
3. Barra de botones de formato. Selecciona un texto ya haz clic para aplicar o bien posiciona el cursor para insertar. Las opciones son (de izquierda a derecha):
    - Encabezados
    - Tamaño de la letra
    - Tipo de letra
    - Negrita (<kbd>CTRL</kbd> + <kbd>B</kbd>)
    - Cursiva (<kbd>CTRL</kbd> + <kbd>I</kbd>)
    - Subrayado (<kbd>CTRL</kbd> + <kbd>U</kbd>)
    - Tachado (<kbd>CTRL</kbd> + <kbd>SHIFT</kbd> + <kbd>X</kbd>)
    - Subíndice
    - Superíndice
    - Marcar texto con colores / Borrar el marcado
    - Alineación del texto
    - Lista numerada
    - Lista no numerada
    - Lista de tareas con casillas
    - Insertar caracter especial
    - Insertar enlace a otra página de Urantiapedia mediante asistente
    - Insertar enlace a cualquier página (<kbd>CTRL</kbd> + <kbd>L</kbd>)
    - Citas
    - Insertar imagen
    - Insertar tabla
    - Estilo código
    - Bloque de código
    - Insertar video
    - Línea de separación
    - Quitar formato (*aún no disponible*)
    - Deshacer (<kbd>CTRL</kbd> + <kbd>Z</kbd>)
    - Rehacer (<kbd>CTRL</kbd> + <kbd>Y</kbd>)


## Functionalidad futura

Algunas acciones importantes del editor aún no están implantadas y se harán en un futuro próximo:
- Inserción de notas al pie de página
- Auto-enlaces en referencias a *El Libro de Urantia* o *La Biblia*.
- Distintos tipos de bloques del estilo de una cita.
- Expresiones matemáticas.
- Diagramas usando el editor `Draw.io`.
