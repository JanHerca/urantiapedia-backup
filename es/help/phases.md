Para cada idioma a incluir en la *Urantipedia* se ejecutará el trabajo en las siguientes fases. Para cada etapa se detalla las tareas a realizar, dónde se realizarán las ediciones y qué colaboradores realizarán cada tarea.

## Fase I - El Libro de Urantia, la Biblia y el Índice de Términos

Tareas | Dónde | Quién
--- | --- | ---
Desarrollo de los conversores de archivos a Wiki.js | GitHub | developers
Convertir cada documento de *El Libro de Urantia* en una página con enlaces a *La Biblia* y a los *Términos* | GitHub | editores GitHub
Convertir cada capítulo de *La Biblia* en una página con enlaces a *El Libro de Urantia* | GitHub | editores GitHub
Desarrollo del *Topic Index Editor*, un asistente que permite acelerar el trabajo con el *Índice de Términos* | GitHub | developers
Conversión del *Índice de Términos* en páginas con enlaces a *El Libro de Urantia* | GitHub | editores GitHub
Carga masiva de los archivos en formato Wiki.js | GitHub | administradores


## Fase II - Libros, artículos e índices

Tareas | Dónde | Quién
--- | --- | ---
Apertura del *Índice de Términos* para su edición manual por los editores | web | editores
Desarrollo para la conversión de libros y artículos a Wiki.js | GitHub | developers
Adición de otros libros que se presuman fuentes de *El Libro de Urantia* aparte de *La Biblia* | GitHub | editores GitHub
Adición de artículos y estudios enlazados con el libro y las fuentes, con las referencias usadas | GitHub | editores GitHub
Adición de índices de personas, seres, lugares, bíblicos, de referencias, de citas, cronologías, etc | GitHub | editores GitHub
Carga masiva de los archivos en formato Wiki.js | GitHub | administradores


## Fase III - Traductor

Tareas | Dónde | Quién
--- | --- | ---
Apertura de los libros y artículos para su edición manual por los editores | web | editores
Desarrollo en Wiki.js un nuevo editor que permita traducciones asistidas directamente en la web | GitHub | developers
Desarrollo en Wiki.js de un modo dual de visualización de *El Libro de Urantia* en dos idiomas | GitHub | developers


## Fase IV - Mapas interactivos

Tareas | Dónde | Quién
--- | --- | ---
Desarrollo en Wiki.js un editor de mapas interactivos, su formato y su renderizador | GitHub | developers
Adición de una colección de mapas interactivos de ejemplo sobre migraciones de razas, hechos históricos y viajes de Jesús de Nazaret | GitHub | editores GitHub


## Fase V - Presentaciones

Tareas | Dónde | Quién
--- | --- | ---
Apertura de los mapas para su edición y adición manual por los editores | web | editores
Desarrollo en Wiki.js de un editor de presentaciones, su formato y su renderizador | GitHub | developers
Adición de una colección de presentaciones usando disponibles en algunos sitios web de asociaciones | GitHub | editores GitHub


## Fase VI - Modelos 3D interactivos

Tareas | Dónde | Quién
--- | --- | ---
Apertura de las presentaciones para su edición y adición por los editores | web | editores
Desarrollo en Wiki.js un editor de modelos 3D interactivos, su formato y su renderizador | GitHub | developers
Adición de una colección de modelos 3D, como la Isla del Paraíso, las esferas arquitectónicas, o el Jardín del Eden | GitHub | editores GitHub

## Futuro

A partir de aquí se podrán evaluar otras posibles adiciones futuras al proyecto. Algunas ideas de cara al futuro que podrían albergarse en la *Urantiapedia*:
* Servir de lugar centralizador para los esfuerzos de traducción del libro a otros idiomas.
* Servir de portal donde ofrecer cursos de formación online sobre diferentes temáticas relacionadas con *El Libro de Urantia*.
