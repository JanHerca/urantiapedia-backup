## Colaboração

A colaboração com o projeto [Jesus Metaverse](https://jesusmetaverse.org/) continua. Começamos a criar uma seção nas _Urantiapedia Tools_ que será capaz de se conectar aos [serviços AirTable](https://www.airtable.com/) (usados ​​pelo projeto Metaverse), para criar um *Urantia Book Database* que beneficiará tanto os projetos quanto qualquer projeto futuro que use informações do _Livro de Urântia_. Este desenvolvimento ainda está nos primeiros passos, e será desenvolvido como um projeto paralelo, então levará algum tempo. Mais notícias aparecerão regularmente sobre este tópico no futuro.
