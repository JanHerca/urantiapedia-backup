## :sparkles: Novos conteúdos

- O site ainda não está disponível. Na parte de conteúdo, continuamos enviando conteúdo diretamente para o [GitHub](https://github.com/JanHerca/urantiapedia).
- Os arquivos de _O Livro de Urântia_ foram gerados em JSON e em Wikitexto. Veja abaixo sobre esses dois formatos.
- Continuamos a incluir referências cruzadas entre a _Bíblia_ e _O Livro de Urântia_, o [Paramony](https://urantia-book.org/urantiabook/paramony/), uma listagem existente criada por Duane Faw, leitor do livro.
- Todo o conteúdo por enquanto está apenas em espanhol.

## :wrench: Urantiapedia Tools

- Vendo a complexidade que as _Urantiapedia Tools_ finalmente terão, decidiu-se iniciar uma migração do código de C# para Javascript, e usar [Electron](https://www.electronjs.org/) como ambiente de execução desktop . Utilizando Javascript, que é uma linguagem muito rápida para codificação, poderemos agilizar todos os desenvolvimentos, e também, por se tratar de um código compatível com web, poderemos reutilizá-lo futuramente na própria página web.
- Adicionado o conversor de formato [LaTeX](https://en.wikipedia.org/wiki/LaTeX) de _O Livro de Urântia_ para um formato intermediário [JSON](https://en.wikipedia.org/wiki/JSON) , o que será mais confortável para lidar com a adição de referências cruzadas a ele.
- Foi adicionado o conversor de _O Livro de Urântia_ do formato JSON para o formato Wikitexto. Este formato é o utilizado pelo [Mediawiki](https://www.mediawiki.org/wiki/MediaWiki), o software da Wikipédia, e que também utilizaremos na Urantiapedia. Este conversor também adiciona notas de rodapé do Paramony no processo.
