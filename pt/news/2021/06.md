Isso nos permitirá traduzir rapidamente essas notas de rodapé para outros idiomas. :globe_with_meridians:

## :sparkles: Novos conteúdos

- Criação no [GitHub](https://github.com/JanHerca/urantiapedia) de um arquivo JSON com notas de rodapé para cada idioma.
- Adição no GitHub da versão em inglês do _O Livro de Urântia_ com notas de rodapé no formato JSON.
