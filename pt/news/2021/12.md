<img src="/_assets/svg/icon-news.svg" alt="News" style="width: 80px;"> A conversão de [Mediawiki](https://www.mediawiki.org/wiki/MediaWiki ) para [Wiki.js](https://js.wiki/) continuou este mês em um bom ritmo. Já temos um novo domínio, .org, por isso estamos lançando uma nova hospedagem para a página, com um novo visual bem diferente do clássico Mediawiki. Há muitas coisas para ajustar nesta nova versão do site, mas a possibilidade de usá-lo em modo multilíngue já começa a tomar forma. O Natal está ao virar da esquina e este próximo ano promete trazer muitas novidades emocionantes para este projeto. :snowflake: :snowman: :santa: :christmas_tree: :gift:

## :sparkles: Novos conteúdos

- Reorganizou todo o conteúdo do projeto no [GitHub](https://github.com/JanHerca/urantiapedia) para remover formatos antigos [Wikitext](https://www.mediawiki.org/wiki/Wikitext/en ) e XML usado em [Mediawiki](https://www.mediawiki.org/wiki/MediaWiki). A partir de agora, o conteúdo para a web estará apenas nos formatos suportados por [Wiki.js](https://js.wiki/), que são HTML e [Markdown](https://www.markdownguide.org/ ).
- Todos os manuais dos contribuidores foram reorganizados e convertidos para o formato Markdown.
- As páginas de boas-vindas foram recriadas em formato HTML para Wiki.js.
- Corrigidos erros no Índice de Tópicos em espanhol e inglês. A capacidade de usar listas numeradas ou não numeradas no texto do tópico foi aprimorada.
- Foi iniciada uma revisão completa de todos os tópicos, começando pelos que começam com um número.

## :wrench: Urantiapedia Tools

- Adicionado um logotipo ao aplicativo e uma seção de configurações que permite usar diferentes temas de estilo.
- Foi desenvolvida uma nova seção no aplicativo, o Editor de Índice de Tópicos, que permite exibir o conteúdo dos tópicos em paralelo em dois idiomas e permite editar o índice de tópicos de forma mais rápida do que através dos arquivos.
