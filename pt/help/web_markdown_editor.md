A captura de tela mostra os controles mais importantes do Editor de Markdown:
1. Título da página
2. Barra de botões Salvar:
    - Botão **Salvar/Salvar**. Se estiver escrito **Salvo**, não há alterações; se estiver escrito **Salvar**, clique para tornar as alterações permanentes. **As páginas não são salvas automaticamente, então não se esqueça de salvar com frequência!** :point_left:
    - Botão **Página**: permite modificar novamente os metadados da página, como título, descrição e tags.
    - Botão **Fechar**: quando terminar a página, feche aqui para voltar ao modo de leitura.
3. Barra de botões de formato. Selecione um texto e clique para aplicar. As opções são (da esquerda para a direita):
    - Audacioso
    - Itálico
    - Tachado
    - Cabeçalhos
    - Subscrito
    - Sobrescrito
    - Cotações e bloqueios
    - Lista não ordenada
    - Lista ordenada
    - Bloco de código
    - Estilo do teclado
    - Linha horizontal
4. Insira a barra de botões. Coloque o cursor onde deseja inserir e clique. As opções são (de cima para baixo):
    - Link para outra página dentro de Urantiapedia. (Para um link para outro site no Markdown, basta colocar a URL, não mais).
    - Imagem: abre o seletor de imagens.
    - Bloco (*ainda não disponível*)
    - Código (*ainda não disponível*)
    - Vídeo e áudio (*ainda não disponível*)
    - Diagramas. Abre o editor `Draw.io` e permite inserir diagramas criados com ele.
    - Expressões matemáticas (*ainda não disponível*)
    - Ajuda com tabelas (*ainda não disponível*)
5. Ajuda de formatação Markdown: Este botão abre uma lista de exemplos de como escrever formatação Markdown.
