Este Editor Visual é um editor WYSIWYG (What You See Is What You Get) baseado no popular [CKEditor 5](https://ckeditor.com/).

Os controles mais importantes do Editor Visual são mostrados na captura de tela:
1. Título da página
2. Barra de botões Salvar:
    - Botão **Salvar/Salvar**. Se estiver escrito **Salvo**, não há alterações; se estiver escrito **Salvar**, clique para tornar as alterações permanentes. **As páginas não são salvas automaticamente, então não se esqueça de salvar com frequência!** :point_left:
    - Botão **Página**: permite modificar novamente os metadados da página, como título, descrição e tags.
    - Botão **Fechar**: quando terminar a página, feche aqui para voltar ao modo de leitura.
3. Barra de botões de formato. Selecione um texto e clique para aplicar ou posicione o cursor para inserir. As opções são (da esquerda para a direita):
    - Cabeçalhos
    - Tamanho da fonte
    - Família de fontes
    - Negrito (<kbd>CTRL</kbd> + <kbd>B</kbd>)
    - Itálico (<kbd>CTRL</kbd> + <kbd>I</kbd>)
    - Sublinhado (<kbd>CTRL</kbd> + <kbd>U</kbd>)
    - Tachado (<kbd>CTRL</kbd> + <kbd>SHIFT</kbd> + <kbd>X</kbd>)
    - Subscrito
    - Sobrescrito
    - Marcar texto com cores / Limpar marcação
    - Alinhamento de texto
    - Lista numerada
    - Lista não numerada
    - Para fazer a lista com caixas de seleção
    - Inserir caractere especial
    - Inserir link para outra página da Urantiapedia usando o assistente
    - Inserir link para qualquer página (<kbd>CTRL</kbd> + <kbd>L</kbd>)
    - Citações
    - Inserir Imagem
    - Insira a tabela
    - Estilo de código
    - Bloco de código
    - Inserir vídeo
    - Linha horizontal
    - Remover formatação (*ainda não disponível*)
    - Desfazer (<kbd>CTRL</kbd> + <kbd>Z</kbd>)
    - Refazer (<kbd>CTRL</kbd> + <kbd>Y</kbd>)


## Funcionalidade futura

Algumas ações importantes do editor ainda não foram implementadas e serão realizadas em breve:
- Inserção de notas de rodapé
- Links automáticos em referências ao *O Livro de Urântia* ou *A Bíblia*.
- Diferentes tipos de blocos no estilo de uma cotação.
- Expressões matemáticas.
- Diagramas usando o editor `Draw.io`.
