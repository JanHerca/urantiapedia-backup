Aqui descrevemos como criar um catálogo de livros que supostamente são fontes do *O Livro de Urântia* ou têm alguma relação ou paralelo com *O Livro de Urântia*.

## Contexto

*O Livro de Urântia* em si oferece duas indicações claras sobre um material humano que foi usado para sua elaboração ([LU 0:12.12](/pt/The_Urantia_Book/0#p12_12) e [LU 121:8.13](/pt/The_Urantia_Book/121#p8_13)). Somado a tudo isso, deve-se notar que *O Livro de Urântia* cita muitas vezes textualmente ou implicitamente livros da *Bíblia*, incluindo livros extra-canônicos e outros.

Vários leitores proeminentes há muito compilam esses paralelos. O leitor Duane L. Faw publicou um extenso volume intitulado [Paramony](https://www.amazon.com/Paramony-Duane-L-Faw/dp/0965197255/ref=sr_1_2?crid=2FCT0LM14FRVT) no qual detalha alguns 60.000 paralelos entre *O Livro de Urântia* e *A Bíblia*. Outro leitor proeminente, Matthew Block, passou anos desenvolvendo [um projeto de localização de paralelos](https://urantiabooksources.com/) com livros não bíblicos escritos até o momento da concepção de *O Livro de Urântia*, tendo detectado mais de uma centena de livros como fontes potenciais dos autores.

## Processo

Esta tarefa é a criação de uma lista de todos os livros não bíblicos que foram detectados com paralelos com *O Livro de Urântia*. Uma fonte inestimável para esta listagem é ir ao site de Matthew Block.

A lista provisória de livros pode ser encontrada aqui: [Livros](/en/book).

Para editar essa lista, você pode fazer isso de duas maneiras:
- Faça login aqui em *Urantiapedia*. Vá para a lista de livros: [Livros](/pt/book). Coloque a página em edição no menu *Ações da página > Editar* no painel de botões superior direito. Um editor Markdown simples será aberto, onde você poderá editar a listagem de livros. (IMPORTANTE: para poder fazer isso, você precisa de um Administrador para habilitar as permissões de edição nessa página.)
- No projeto GitHub, modifique este arquivo: `/output/wikijs/en/book.md`. Para modificá-lo, você pode simplesmente baixar esse arquivo ou todo o projeto do GitHub. Veja a [ajuda dos Editores Assistentes](/pt/help/github_assistant) para saber como baixar coisas do GitHub.

Em qualquer um dos dois casos anteriores, o arquivo a ser editado é um arquivo Markdown muito simples onde para cada livro você deve incluir o título, o autor e o ano da primeira edição.

Se você modificou a listagem do arquivo no GitHub, você tem duas opções para enviar suas alterações: 1) se você for um “editor chefe” use o GitHub conforme detalhado em [ajuda do editor chefe](/pt/help/github); se você for um “Editor Assistente”, você pode simplesmente enviar esse arquivo Markdown modificado para o e-mail de qualquer Administrador ou para urantiapedia@gmail.com.
