## I. LA NATURE DU PARADIS

1. Le Père Universel a révélé le potentiel infini de son moi non personnel en tant que Paradis. <a id="s29_96"></a>[LU 11:9.3](/fr/The_Urantia_Book/11#p9_3)
2. Le paradis n'est pas sphérique, il est ellipsoïde. <a id="s30_54"></a>[LU 11:2.2](/fr/The_Urantia_Book/11#p2_2)
3. Le temps et l'espace sont inexistants au Paradis. <a id="s31_53"></a>[LU 0:1.13](/fr/The_Urantia_Book/0#p1_13)
4. Le paradis est le centre de l'univers des univers. <a id="s32_54"></a>[LU 11:0.1](/fr/The_Urantia_Book/11#p0_1)
5. Le paradis est le centre géographique de l'infini. <a id="s33_54"></a>[LU 11:9.2](/fr/The_Urantia_Book/11#p9_2)
6. Le Paradis est immobile – c'est l'Absolu du contrôle de la gravité matérielle. <a id="s34_82"></a>[LU 0:4.12](/fr/The_Urantia_Book/0#p4_12)
7. Le paradis est composé d'absolutum. <a id="s35_39"></a>[LU 11:2.9](/fr/The_Urantia_Book/11#p2_9)
8. Le paradis est l'absolu des modèles. <a id="s36_40"></a>[LU 11:9.5](/fr/The_Urantia_Book/11#p9_5)
9. Le Paradis est divisé en trois domaines :
	> 1. Haut Paradis.
	> 2. Paradis périphérique.
	> 3. Nether Paradise. <a id="s40_23"></a>[LU 11:2.4](/fr/The_Urantia_Book/11#p2_4)
10. Le paradis n'est pas un créateur, c'est un contrôleur unique. <a id="s41_66"></a>[LU 0:4.13](/fr/The_Urantia_Book/0#p4_13)
11. Le Paradis n'est pas la Déité—il n'est pas conscient. <a id="s42_58"></a>[LU 11:9.4](/fr/The_Urantia_Book/11#p9_4)

## II. LA RÉSIDENCE DIVINE

1. Le paradis est la demeure du Dieu éternel. <a id="s46_46"></a>[LU 0:0.5](/fr/The_Urantia_Book/0#p0_5)
2. Le Père Universel est cosmiquement focalisé, spirituellement personnalisé et géographiquement résident au Paradis. <a id="s47_118"></a>[LU 11:1.2](/fr/The_Urantia_Book/11#p1_2)

## III. HAUT PARADIS

1. Le Paradis supérieur se compose de trois sphères :
	> 1. Présence de la Déité.
	> 2. Sphère Très Sainte.
	> 3. Aire Sainte. <a id="s54_19"></a>[LU 11:3.1](/fr/The_Urantia_Book/11#p3_1)
2. Chacune des nombreuses unités résidentielles héberge un milliard de groupes de travail. <a id="s55_91"></a>[LU 11:3.4](/fr/The_Urantia_Book/11#p3_4)
3. L'adoration est la plus grande joie des habitants du Paradis. <a id="s56_65"></a>[LU 27:7.5](/fr/The_Urantia_Book/27#p7_5)
4. Le paradis est le destin de tous les mortels ascendants. <a id="s57_60"></a>[LU 11:9.1](/fr/The_Urantia_Book/11#p9_1)

## IV. PARADIS PÉRIPHÉRIQUE

1. Sur le Paradis périphérique se trouvent les champs d'atterrissage et d'expédition ; également le siège central de la force des Sept Maîtres Esprits. <a id="s61_152"></a>[LU 11:4.1](/fr/The_Urantia_Book/11#p4_1)
2. Voici les zones d'exposition historiques et prophétiques. <a id="s62_61"></a>[LU 11:4.3](/fr/The_Urantia_Book/11#p4_3)

## V. LE PARADIS DU NETHER

1. Au bas du Paradis, tous les circuits d'énergie physique ont leur origine. <a id="s66_77"></a>[LU 11:5.1](/fr/The_Urantia_Book/11#p5_1)
2. Voici la présence spatiale de l'Absolu Non Qualifié. <a id="s67_56"></a>[LU 11:5.7](/fr/The_Urantia_Book/11#p5_7)
3. Au bas du Paradis se trouve le centre de la respiration spatiale. <a id="s68_69"></a>[LU 11:5.5](/fr/The_Urantia_Book/11#p5_5)
4. Le paradis existe sans le temps et n'a pas de place dans l'espace. <a id="s69_70"></a>[LU 11:2.10](/fr/The_Urantia_Book/11#p2_10)
5. L'espace est un don du Paradis. <a id="s70_35"></a>[LU 11:7.4](/fr/The_Urantia_Book/11#p7_4)
