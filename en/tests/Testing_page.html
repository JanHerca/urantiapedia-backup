Testing procedure

Here we are testing:

  1. Creating a page in HTML through GitHub
  2. Edit is in the web
  3. Check that a duplicate is create outside EN folder
  4. Copy duplicated content inside old file
  5. Remove the duplicate and commit
  6. Commit the updated
  7. Check if history is maintained

Another alternative to remove + commit + commit update could be to remove and commit update at the same time.
