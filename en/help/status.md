## Milestone I - The Urantia Book, the Bible and Topic Index

For each language supported the list of tasks to complete milestone I is this:

1. Translation of _Paramony_ from English to the target language (`Paramony`).
2. Translation and review of _Topic index_ from English to the target language (`Topics`).
3. Obtain a public-domain translation of _The Bible_ in the target language (`Bible`).
4. Conversion _The Urantia Book_ from HTML to JSON without _Paramony_ footnotes (`UB`).
5. Addition of _Paramony_ footnotes to _The Urantia Book_ in JSON (`UB + Paramony`).
6. Conversion of the _Topic index_ from TXT to Wiki.js and commit changes using [Urantiapedia-backup](https://github.com/JanHerca/urantiapedia-backup) (`Topics`).
7. Conversion of _The Urantia Book_ from JSON with _Paramony_ footnotes to Wiki.js including references to topics, and commit changes using [Urantiapedia-backup](https://github.com/JanHerca/urantiapedia-backup) (`UB Wiki`).
8. Conversion of _The Bible_ from LaTeX to Wiki.js with _Paramony_ footnotes, and commit changes using [Urantiapedia-backup](https://github.com/JanHerca/urantiapedia-backup) (`Bible Wiki`).

## Planned tasks

Here you can find a list of tasks that currently are not active. If you are a contributor or want to be one check this list frequently to see where you can do your best additions. If a task is indented under above that means that the task depends on previous to start.

- The Urantia Book :blue_book: :
  - Automatic translation of _Paramony_ footnotes: `id`, `ja`, `ko`, `lt`, `nl`, `pl`, `pt`, `ru`, `sv`.
    - Review of previous translation: `bg`, `cs`, `da`, `de`, `el`, `et`, `fa`, `fi`, `fr`, `he`, `hu`, `it`, `ro`.
- The Bible :closed_book: :
  - Obtain a public domain Bible: `et`, `fi`, `fr`, `he`, `hu`, `id`, `ja`, `ko`, `lt`, `pl`, `pt`, `ro`, `ru`, `sv`.
  - Automatic translation of Paramony footnotes: all languages apart from `bg`, `en` and `es`.
    - Review of previous translation: all languages apart from `en` and `es`
- The Topic Index :card_index: :
  - Produce the first version for `en` and `es`. This is a heavy task that has been divided by letters of the alphabet. Letters `A` to `G` are already finished and remain letters `H` to `Z`.
    - Automatic translation of Topic Index: it is not recommended to start this tasks until previous is completed because some automatic process will help here.
      - Review of previous translation: as above it is not recommended to start this task until produce first version for `en` and `es`.
- Books :books: :
  - Obtain public domain Extra-Biblical books: all languages.
  - Obtain books supposed sources (downloading from web or scanning): books are usually in `en`.
    - Automatic translation of previous books: all languages apart from `en`.
      - Review of previous translation: all languages apart from `en`.
- Articles :page_with_curl: :
  - Automatic translation of articles: all languages apart from `en`.
    - Review of previous translation: all languages apart from `en`.
- Study aids :notebook: :
  - Plan a list of study aids to include.
- Schemas :memo: :
  - Plan a list of schemas to include.
- Indexes :capital_abcd: :
  - Plan a list of indexes to include.
- Maps :earth_americas: :
  - Create a geodatabase and QGIS projects for auto generation of maps in SVG, HTML image maps (static maps), and KML (interactive maps).
    - Automatic translation of texts in maps: all languages.
      - Review of previous translations: all languages.
- Audio :sound: :
  - Test best suitable text-to-speech solutions: all languages.
  - Generation of audio of The Urantia Book: currently available in `en` and in process with `es`.
- Images :sunrise_over_mountains: :
  - Create an image catalog for illustrating The Urantia Book, the Bible and the Topic Index. Currently is done with part IV of Urantia Book.
- Video :movie_camera: : 
  - Create a extended introduction to Urantiapedia (Youtube channel).
  - Create an advanced tutorial for Urantiapedia contributors (Youtube channel).
