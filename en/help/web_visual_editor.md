This Visual Editor is a WYSIWYG (What You See Is What You Get) editor based on the popular [CKEditor 5](https://ckeditor.com/).

The most important controls of the Visual Editor are shown in the screenshot:
1. Page Title
2. Save button bar:
    - **Saved / Save** button. If it says **Saved**, there are no changes; if it says **Save** click to make the changes permanent. **Pages don't auto-save so don't forget to save frequently!** :point_left:
    - Button **Page**: allows you to re-modify the metadata of the page, such as the title, description, and tags.
    - **Close** button: when you are done with the page, close here to return to reading mode.
3. Format button bar. Select a text and click to apply or position the cursor to insert. The options are (from left to right):
    - Headers
    - Font size
    - Font family
    - Bold (<kbd>CTRL</kbd> + <kbd>B</kbd>)
    - Italics (<kbd>CTRL</kbd> + <kbd>I</kbd>)
    - Underline (<kbd>CTRL</kbd> + <kbd>U</kbd>)
    - Strikethrough (<kbd>CTRL</kbd> + <kbd>SHIFT</kbd> + <kbd>X</kbd>)
    - Subscript
    - Superscript
    - Mark text with colors / Clear markup
    - Text alignment
    - Numbered list
    - Unnumbered list
    - To do list with check boxes
    - Insert special character
    - Insert link to another Urantiapedia page using wizard
    - Insert link to any page (<kbd>CTRL</kbd> + <kbd>L</kbd>)
    - Quotes
    - Insert image
    - Insert table
    - Code style
    - Code block
    - Insert video
    - Horizontal line
    - Remove formatting (*not yet available*)
    - Undo (<kbd>CTRL</kbd> + <kbd>Z</kbd>)
    - Redo (<kbd>CTRL</kbd> + <kbd>Y</kbd>)


## Future functionality

Some important actions of the editor are not implemented yet and will be done in the near future:
- Insertion of footnotes
- Auto-links in references to *The Urantia Book* or *The Bible*.
- Different types of blocks in the style of a quote.
- Mathematical expressions.
- Diagrams using the `Draw.io` editor.
